# 회사가 운용되기 위해 필요한 솔루션
컴퓨터 그래픽스를 주로 다루는 게임, VFX, 애니메이션 관련 회사들도 일반적인 회사와 크게 다르진않습니다.

콘텐츠를 제작하는 회사에서 필요한 리스트입니다. 우리는 SW 영역을  주제로 다루며 강의를 진행합니다.
